vscode 사용 팁

1. class 더블 클릭시 - 기호 때문에 class 전체가 선택 안되는 번거로움이 있으므로, ctrl + , > editor.wordSeparators 에서 - 기호는 빼면 편함.
2. 설정에서 detect indentation 체크하면 원하는 부분에 범위 지정 후 (ctrl + k) + f로 코드 포맷시 해당 파일의 들여쓰기에 맡게 알아서 포맷되어 편리함.
3. 우측 상단에 '...' 기호 클릭 > [편집기 열어 두기] 체크하면 파일 열때 새창으로 열림.

ebs 중학 영어 문법
